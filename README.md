# halcyon

bare bones web app foundation for the game `halcyon`.

## run

open `index.html` in a browser, or serve this folder with any static file server.

## notes

- full-window web app
- background color: `#E8DCC6`
- gameplay border color: `#D8E2D0`
- soft inner glow color: `#FFFDF4`
- player-facing text is lowercase
- cursor locks after clicking the gameplay area and stays bounded inside the border
